# Recura — Modern SaaS Dashboard

Recura is a professional SaaS dashboard built with Next.js, React, Tailwind CSS, and TypeScript. It features a clean, modular architecture and a robust design system.

## Getting Started

1.  **Extract Files**: Unzip the project folder.
2.  **Install Dependencies**: Run `npm install`.
3.  **Run Development Server**: Run `npm run dev`.
4.  **Open Project**: Navigate to `http://localhost:4000`.

## Tech Stack

-   **Framework**: Next.js 16
-   **Library**: React 19
-   **Styling**: Tailwind CSS 4
-   **Language**: TypeScript
-   **Charts**: Recharts
-   **Theming**: next-themes

## Project Structure

-   `src/app`: Next.js App Router
-   `src/components`: Global reusable UI components
-   `src/modules`: Feature-based modules
-   `src/lib`: Helpers and utilities

## Payment Support

-   Stripe (Ready)
-   Paystack (Extendable)

## Customization

Colors and branding can be easily modified via the Tailwind configuration and the branding settings.

## Deployment

Recommended to deploy on Vercel or any Node.js server.
